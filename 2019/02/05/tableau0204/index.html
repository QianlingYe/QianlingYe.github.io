Joining Data

  2. Joins with duplicated values

Blending Data

  1. A blend is like a smart join “on the fly”
  2. It is a left join
  3. Common fields with the same name are picked up automatically as the blend cause
  4. You can control this via the “Data” menu
    5. Blending occurs at the granularity of your view
    6. Aggregation happens before the blend
    7. Blends are unique to each worksheet

Joining vs Blending

Use Joins when combining data at row level
Use Blends when Datasources have different levels of granularity；Datasources come from different systems
